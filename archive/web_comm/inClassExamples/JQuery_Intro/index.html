What is JQuery?

JQuery is a Javascript Library specifically written manipulating DOM elements (the elements that make up an HTML document) and changing their css/display properties. Its syntax of selectors and actions makes it very human-readable and easy to select elements on the page by their class, id, or selector and do things to them.

When using JQuery, remember you should always reference the The JQuery Library in your page, before you write any scripts.

The jQuery syntax is tailor-made for selecting HTML elements and performing some action on the element(s).

Basic syntax is: $(selector).action()

A $ sign to define/access jQuery
A (selector) to "query (or find)" HTML elements
A jQuery action() to be performed on the element(s)

Examples:
$(this).hide() - hides the current element.
$("p").hide() - hides all p elements.
$(".test").hide() - hides all elements with class="test".
$("#test").hide() - hides the element with id="test".

The .toggle() function lets you click between hiding and showing. Click the button above to hide this thing, and click it again to show this thing.

You can also get more specific with how things are displayed with specific action functions like .slide. This div will slideToggle() when the button is clicked.

You can also use JQuery to add, remove or toggle a class in CSS. That way, you can assign styles to be manipulated with user-interaction. You can use this to change color, background, display properties: anything that you can style with CSS.
