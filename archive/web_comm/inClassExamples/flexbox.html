Flexbox: Examples

Flexbox is a kind of display mode that helps to automatically align and space content. It's easy to learn and pretty intuitive. It's also a good way to incorporate some simple "responsive" behavior for elements to lay out well on large and small screens.
Flexbox works on parent-child relationships. The parent container is given the display property "flex", which enables flex content for all of its DIRECT children. See the simple example below. "margin:auto" on the .flexbox1_child element evenly spaces the three divs with equal margins.

By default, flex items try to stay on one line. Setting the "flex-wrap" property allows content to wrap onto multiple lines when sized down. You set this property on the container or parent element, and you can set it to "wrap", "nowrap" or "wrap-reverse" which will reverse the direction of the content on the next line.

Setting the "justify-content" property helps to space the elements evenly. The divs below are set to "justify-content: space-around".
