There are tons of elements in bootstrap that are designed for web-standard communication forms (blogs, profiles, Wikis, etc.). You can utilize the functionality of these, and change the styling, for your own design.

One of my favorite bootstrap utilities is "collapse", which lets you tie hide/show content to buttons and accordion menus very easily.

More about Collapse

Simple Collapsible

Simple collapsible
the href of the button is "#demo", and the id of the collapsable content is "demo"; the data-toggle attribute is set to "collapse".

You can also make a whole accordion menu!

Accordion

Note: The data-parent attribute makes sure that all collapsible elements under the specified parent will be closed when one of the collapsible item is shown.

Collapsible Group Item #1
this one has the class "show" and so it will be showing when the page loads. Take this class off, or add it to a different div in the menu to see this in action.
Collapsible Group Item #2
pay close attention as well to the data-parent, data-toggle and id attributes. The ids are linked to the hrefs of each Item heading. The data-parent and data-toggle attributes work with the underlying bootstrap javascript to help these elements open and close. Each of the collapse bodies must have the data-parent accordion, and the data-toggle "collapse" in order to work.
